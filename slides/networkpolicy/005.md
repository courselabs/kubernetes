# Ingress Rules (Incoming Traffic)

```mermaid
graph LR
    SOURCE1[<carbon-container-software/> Pod<br/>app=frontend] -->|allowed| TARGET[<carbon-container-software/> Pod<br/>app=backend]
    SOURCE2[<carbon-container-software/> Pod<br/>app=other] -.->|denied| TARGET

    NP[<carbon-network-overlay/> NetworkPolicy<br/>ingress from: app=frontend]
    NP -.->|protects| TARGET

    style SOURCE1 fill:#4ade80
    style SOURCE2 fill:#ef4444
    style TARGET fill:#fbbf24

```

<div class="mt-8 text-center">

### `spec.ingress` controls who can connect TO the protected Pods

```yaml
ingress:
- from:
  - podSelector:
      matchLabels:
        app: frontend
  ports:
  - port: 8080
    protocol: TCP
```
