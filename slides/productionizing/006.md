# Startup Probe

```mermaid
graph TB
    START[<carbon-rocket/> App Starting] --> STARTUP[<carbon-time/> Startup Probe]

    STARTUP -->|Checking...| BLOCK[<carbon-locked/> Liveness & Readiness<br/>blocked]

    STARTUP -->|Success| ENABLE[<carbon-unlocked/> Enable other probes]

    style START fill:#60a5fa

    style BLOCK fill:#ef4444
```

<div class="mt-8 text-center text-sm">

### For slow-starting apps
```yaml
startupProbe:
  httpGet:
    path: /started
    port: 8080
  failureThreshold: 30
  periodSeconds: 10
```

<div class="mt-4 opacity-80">
Prevents premature liveness restarts
