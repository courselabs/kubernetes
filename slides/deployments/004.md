# Rapid Creation

<carbon-flash class="text-5xl text-yellow-400 mb-4" />

## Imperative Commands

```bash
kubectl create deployment \
  nginx \
  --image=nginx \
  --replicas=3
```

<carbon-terminal class="text-5xl text-green-400 mb-4 mt-6" />

## Generate & Edit

```bash
kubectl create deployment ... \
  --dry-run=client \
  -o yaml > deploy.yaml
```
