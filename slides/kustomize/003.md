### Kustomize manages configuration variants

<div class="grid grid-cols-2 gap-8 mt-8">
<div>

**Without Kustomize (CKAD)**
- Copy YAML files
- Modify manually
- Apply with kubectl

</div>
<div>

**With Kustomize**
- Define base once
- Apply overlays
- Use kubectl apply -k
