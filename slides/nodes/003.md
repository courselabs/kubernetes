# Essential Node Commands

```mermaid
graph TB
    CMDS[<carbon-terminal/> Node Commands]

    CMDS --> GET[kubectl get nodes<br/>-o wide, --show-labels]
    CMDS --> DESCRIBE[kubectl describe node<br/>capacity, conditions]
```

### Get nodes: Status, roles, age
### Describe: Detailed capacity and conditions
### Top: Current resource usage (requires metrics-server)
