# Update Strategies

```mermaid
graph LR
    subgraph RollingUpdate
        R[<carbon-renew/> RollingUpdate<br/>Default] --> AUTO[<carbon-flash/> Automatic<br/>Updates]
    end

    subgraph OnDelete
        O[<carbon-trash-can/> OnDelete] --> MANUAL[<carbon-user/> Manual<br/>Deletion Required]
    end

    style R fill:#60a5fa
    style AUTO fill:#4ade80
    style O fill:#fbbf24
    style MANUAL fill:#f97316
```

<div class="mt-8 text-center">

### RollingUpdate (default)
Pods automatically updated when DaemonSet changes

### OnDelete
Pods only updated when manually deleted

<div class="mt-4 opacity-80">
Configure `maxUnavailable` for rolling updates
