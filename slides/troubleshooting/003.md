```mermaid
graph LR
    START[<carbon-warning-alt/> Issue] --> STATUS[<carbon-terminal/> kubectl get] --> DESCRIBE[<carbon-search/> kubectl describe] --> LOGS[<carbon-document/> kubectl logs] --> FIX[<carbon-edit/> Fix & Verify]

    style START fill:#ef4444
```

<div class="mt-8 text-center">

### Always follow the same workflow
1. Check status
2. Describe to see events
3. Check logs if running
4. Fix issue
5. Verify resolution
